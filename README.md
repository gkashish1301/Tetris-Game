# 🕹️ JavaScript Tetris

A modern recreation of the classic arcade game, Tetris. This project focuses on smooth block movement, collision detection, and efficient line-clearing logic.

## 🎮 How to Play
* **Move:** Use the **Arrow Keys** (Left/Right) to position the falling pieces.
* **Rotate:** Use the **Up Arrow** to rotate the blocks to fit into gaps.
* **Drop:** Use the **Down Arrow** to speed up the fall.
* **Goal:** Complete horizontal lines to clear them and increase your score. The game ends if the blocks reach the top of the screen!

## ✨ Key Features
* **Randomized Spawning:** All 7 classic Tetris shapes (Tetrominoes).
* **Collision Detection:** Prevents blocks from moving through walls or other pieces.
* **Score Tracking:** Earn points for every line cleared.
* **Organized Code:** Modular structure with separate folders for `Scripts` and `CSS`.

## 🛠️ Tech Stack
* **HTML5 Canvas:** For rendering the game grid.
* **JavaScript:** For the game engine and input handling.
* **CSS3:** For the retro gaming aesthetic.

## 📂 Project Structure
* `/Scripts`: Contains the core game logic and movement functions.
* `/CSS`: Contains the styling for the game board and UI.
* `index.html`: The main entry point for the game.
